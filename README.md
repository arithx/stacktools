# stacktools

Python tool for creating and testing OpenStack environments.


Commands
========

``devstack-deploy``
-------------------
::

    devstack-deploy [-l/--localrc] [-v/--virt-driver] [-b/--backend] [-t/--libvirt-type]

``tempest-deploy``
------------------
::
    tempest-deploy [-p/--path]

``tempest-run``
---------------
::
    tempest-run [-r/--regex] [-c/--concurrency]
